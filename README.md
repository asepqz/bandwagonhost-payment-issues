## 搬瓦工付款错误是什么？为什么会发生

**搬瓦工付款错误**，简单说就是在购买 VPS 套餐时，付款流程出现中断或异常，导致钱扣了但服务没开通，或者订单直接被系统拒绝。

搬瓦工后台用的是 WHMCS 系统，内置了一套风控模块（Fraud Protection）。只要你下单时的 IP 地址和你填写的注册国家不一致，系统就会自动标记为 Fraud 并退款取消订单。这套逻辑设计上是为了防止刷单，但对国内用户来说挺容易踩——尤其是那种习惯挂着代理浏览网页的人，一不注意就中招了。

支付宝付款失败则是另一回事，通常是搬瓦工那边的支付宝接口出了问题，跟你本地网络关系不大。

## 问题一：订单状态变成 Fraud——钱退了但 VPS 没了

这是搬瓦工付款错误里最让人懵的一种。你付款成功，VPS 也开着用，结果第二天早上收到一封邮件，主题是"Invoice Refund Confirmation"，VPS 被自动取消，钱原路退回。

**为什么会这样？**

1. 下单时开着 VPN 或代理。你的 IP 显示在美国，但注册信息写的是中国，系统判定不匹配，触发风控。
2. 用了不正规的支付账号。PayPal 账户或信用卡来源有问题，事后被搬瓦工补查发现。
3. 用自己的推广链接注册了小号。搬瓦工会识别这种操作，直接打 Fraud 标记。
4. 同一 IP 频繁注册新账户然后申请退款。被识别为刷退款行为。

**怎么解决？**

关掉所有代理工具，用真实网络环境重新下单。注册信息里的国家选中国，其他信息可以不完全真实，但不要写得明显离谱——你在广州就别写纽约。邮箱必须写对，因为激活邮件和退款通知都发到这里。

重新下单后，正常情况下不会再触发 Fraud。如果反复被判，可以登录账户提交工单联系客服，说明情况请求人工审核，通常几个工作日内有回复。

## 问题二：支付宝付了钱，订单显示未支付（unpaid）

这种情况比较常见，大多数时候不是你的问题。

付款流程是这样的：你在支付宝那边完成付款 → 支付宝通知搬瓦工 → 搬瓦工确认收款 → 订单状态变 paid。如果这条通知链路中间断了，你那边钱扣了，搬瓦工这边却没收到确认，订单就一直挂着 unpaid。

**分两种情况：**

一、网页提前关掉了或者跳转失败

付完支付宝，返回搬瓦工页面的跳转有时候会卡。这种情况下，支付宝其实已经扣款成功，你不需要重新付。登录搬瓦工账户，首页就能看到未支付的账单，找到那笔，点"View Invoice"，然后换一种支付方式（信用卡或 PayPal）补付即可——钱不会重复扣，因为这笔账单已经在支付宝那边有记录，你补付的是让搬瓦工系统识别到付款状态。

等等，听起来很绕？其实只要记住一件事：**付款失败了不要立刻重新下单，先去账户里找那张未支付的账单，在账单页换方式付就行。**

二、PayPal 接口故障

搬瓦工历史上出现过几次 PayPal 支付接口问题，付款成功后订单依然 unpaid。遇到这种情况，官方的处理方式是：登录 PayPal，找到那笔交易，复制 Transaction ID（交易号），然后在搬瓦工后台提交工单，把交易号附上，客服会手动确认并激活订单。

## 问题三：支付宝直接报错，SECONDARY_MERCHANT_STATUS_ERROR

这条错误信息是搬瓦工支付宝接口的问题，不是你账号有什么异常。出现这个错误时，支付宝那边都不会跳出付款页面，直接就提示"系统有点儿忙，一会儿再试试"。

解决方案：

1. **等一等再试。** 搬瓦工出现这个问题通常会在几个小时到一两天内自行恢复。
2. **换支付方式。** 直接在账单页面切换到银联支付（UnionPay via Payssion），支持云闪付扫码，对国内用户来说很方便。也可以用 PayPal 或者信用卡。
3. **账单已经生成了别再重复下单。** 只要你走到了填信息那一步，账单就已经生成在你账户里了，换方式付那张已有的账单就好，不需要从头再来一遍流程。

## 搬瓦工支持的付款方式汇总

目前搬瓦工支持四种付款方式，国内用户几乎都能用上：

- **支付宝（Alipay）** — 最常用，直接扫码付款
- **银联支付（UnionPay via Payssion）** — 支持云闪付，国内银行卡都行
- **PayPal** — 需要提前注册，可绑定国内银联卡
- **信用卡** — Visa / MasterCard 均可

注意：搬瓦工账户余额充值不支持支付宝，只能用 PayPal 或信用卡。但直接购买套餐是支持支付宝的，两者逻辑不同。

## 问题四：付款成功了，但 VPS 一直没激活

搬瓦工大多数套餐付完款后几分钟内自动激活，登录 KiwiVM 控制面板就能看到 VPS 状态变为 running。如果付款已经 15-20 分钟了还没激活，先检查：

1. 邮件里有没有收到激活通知——有时候激活成功了但你没留意
2. 登录账户看服务列表，有时候界面需要刷新
3. 如果确实没有，提交工单说明账单号，客服一般响应比较快，二十几分钟内有回复
